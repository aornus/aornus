随机事件与概率

基本概念

随机现象与样本空间

随机试验

把对自然现象的一次观察称为 试验，其特点如下

  1. 相同条件下可重复进行

  2. 每次结果不可预知

  3. 所有可能结果已知

[易错例子：在纸上随意写三个数字]{style=“color: red”}（结果可知，[但不可重复]{.ul}，）

样本点

随机试验的每一个可能结果称为一个样本点，记为ω\omegaω

样本空间

全体样本点构成的集合，记为Ω\OmegaΩ

事件及其运算

随机事件

：代表试验的每一个可能结果（即Ω\OmegaΩ中的部分样本点组成的结果），是不可再分的事件（基本事件）,如若某个事件每次试验都发生则称之为必然事件记作Ω\OmegaΩ，同理每次都不发生的事件称为不可能事件，记作∅\varnothing∅。

事件的关系

：事件间的关系就是集合之间的关系。一些基本关系如下：

  1. 并：A∪B;A+B,A\cup B;A+B,\qquadA∪B;A+B,
    ⋃i=1∞Ai\displaystyle \bigcup^{\infty}_{i=1} A_ii=1⋃∞​Ai​表示A1,A2,⋯A_1,A_2,\cdotsA1​,A2​,⋯至少有一个发生。

  2. 交：A∩B;A×B,A\cap B;A\times B,\qquadA∩B;A×B,⋂i=1∞Ai\displaystyle \bigcap^{\infty}_{i=1} A_ii=1⋂∞​Ai​表示A1,A2,⋯A_1,A_2,\cdotsA1​,A2​,⋯全部都发生。

  3. 逆：Aˉ (A不发生),A⋅Aˉ=∅,A+Aˉ=Ω\bar{A} \ \text{(A不发生)},\qquad A\cdot\bar{A}=\varnothing,A+\bar{A}=\OmegaAˉ (A不发生),A⋅Aˉ=∅,A+Aˉ=Ω

  4. 差：A−B (A发生且B不发生)A-B \ \text{(A发生且B不发生)}A−B (A发生且B不发生)

  5. 包含A∈BA\in BA∈B，相等A=BA=BA=B

  6. 互不相容事件：A与B不同时发生

  7. 对立事件：A与B不同时发生，且A∪B=ΩA\cup B = \OmegaA∪B=Ω，显而易见前者更宽松，范围更大。

事件间的运算规律

设A，B，C为事件，则有以下规律:

  1. 交换律: A∪B=B∪AAB=BAA\cup B=B\cup A\qquad AB=BAA∪B=B∪AAB=BA

  2. 结合律: (A∪B)∪C=A∪(B∪C)(AB)C=A(BC)(A\cup B)\cup C=A\cup (B\cup C)\qquad (AB)C=A(BC)(A∪B)∪C=A∪(B∪C)(AB)C=A(BC)

  3. 分配律: A∩(B−C)=AB−ACA∩(B∪C)=AB∪ACA\cap(B-C)=AB-AC \quad A\cap (B\cup C)=AB\cup ACA∩(B−C)=AB−ACA∩(B∪C)=AB∪AC

  4. 对偶律:
    A∪B‾=Aˉ∩BˉA⋅B‾=Aˉ+Bˉ\overline{A\cup B}=\bar{A} \cap \bar{B} \qquad \overline{A\cdot B}=\bar{A}+\bar{B}A∪B=Aˉ∩BˉA⋅B=Aˉ+Bˉ

频率与概率

频率

在相同条件下进行了n次试验，若事件A在n次试验中发生了m次，则称事件A在这n此试验中发生的频率为$$F_n(A)=\frac{m}{n}$$

性质

  1. 非负性：0≤Fn(A)≤10\le F_n(A)\le 10≤Fn​(A)≤1

  2. 规范性：Fn(Ω)=1F_n(\Omega)=1Fn​(Ω)=1

  3. 有限可加性：Fn(A∪B)=Fn(A)+Fn(B)F_n(A\cup B)=F_n(A)+F_n(B)Fn​(A∪B)=Fn​(A)+Fn​(B)

  4. 稳定性： 随着n的增大，随机事件的频率趋于稳定

概率

随机试验的次数充分大时，事件A发生的频率稳定在某数P附近摆动，则称p为事件A发生的统计概率，记为$$
P(A)=P $$

性质

  1. 非负性：0≤P(A)≤10\le P(A)\le 10≤P(A)≤1

  2. 规范性：P(Ω)=1P(\Omega)=1P(Ω)=1

  3. 有限可加性：P(A∪B)=P(A)+P(B)P(A\cup B)=P(A)+P(B)P(A∪B)=P(A)+P(B)

古典概型

古典模型的定义

对于某一个随机试验，如果它的全体基本事件是有限个，而且具有等可能性，则称之为古典概型。则对于任意事件A,对应的概率（古典概率）为：$$P(A)=\frac{\text{事件A包含的基本事件数k}}{\text{基本事件总数n}}$$
